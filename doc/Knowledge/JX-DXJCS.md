# 雕铣机常用资料

## 常用刀具

### 玉米铣刀

主要用于耐高温塑料，木 头等非金属材料的铣槽
* 刀柄直径：3.175mm
* 刀刃直径：0.6-3.175mm

### 单刃铣刀

用于塑料、铜、铝合金等 材料的切割，铣边
* 刀柄直径：3.175mm
* 刀刃直径：1、2、3mm
  
### 三刃铣刀

加工铝合金、 铜等材料，铣槽，铣边， 也可以用于非金属电木， 塑料等加工
* 刀柄直径：4、6mm
* 刀刃直径：1、1.5、2、2.5、3、3.5、4、5、6mm
