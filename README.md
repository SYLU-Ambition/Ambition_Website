## 🚀 项目简介
沈阳理工大学电子技术与应用协会Ambition战队官方门户网站，采用VitePress静态站点生成器搭建，支持以下核心特性：

- **响应式布局**：自动适配PC/手机/平板设备
- **文档即代码**：Markdown原生支持 + Vue组件嵌入能力
- **极速构建**：Vite驱动的毫秒级热更新（开发模式）
- **自动化部署**：GitHub Actions实现持续集成
- **超低维护成本**：目前维护成本有且仅有域名的购买费用
